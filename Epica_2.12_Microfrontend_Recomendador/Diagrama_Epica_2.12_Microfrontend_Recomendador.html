HU 2.12.1 — Web component / micro-frontend

El Recomendador se integra como Custom Element reutilizable en el portal CCB, con lazy load, responsive y seguridad anti-XSS.

HU 2.12.2 — Navegación integrada con asistente

El usuario se mueve fluidamente entre el asistente conversacional y el Recomendador con sesión y contexto compartidos.
