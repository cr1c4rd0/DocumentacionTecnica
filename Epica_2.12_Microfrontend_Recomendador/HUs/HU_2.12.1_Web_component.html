Historia de Usuario · HU 2.12.1

Web component / micro-frontend

Enunciado

COMO

desarrollador del portal CCB que integra el recomendador

QUIERO

un Custom Element reutilizable que pueda incrustar en cualquier página del portal

PARA

integrar el recomendador sin acoplar su código al portal base y facilitar actualizaciones independientes

Criterios de Aceptación
  • Custom Element <ccb-recomendador> compatible con cualquier framework
  • Lazy loading: carga solo cuando entra en viewport
  • Responsive con breakpoints consistentes con el portal
  • Parámetros configurables via atributos HTML (userId, canal, límite)
  • Sanitización de outputs anti-XSS
  • Isolated styles con Shadow DOM o CSS scoping
  • Tiempo de carga inicial ≤ 2 segundos
Notas Técnicas
  • Web Components estándar (Custom Elements v1)
  • Bundle independiente versionado
  • CDN propio para distribución
  • Postmessage API para comunicación con el portal padre
  • CSP headers compatibles
