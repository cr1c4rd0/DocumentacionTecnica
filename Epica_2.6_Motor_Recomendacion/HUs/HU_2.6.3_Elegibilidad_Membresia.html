Recomendaciones por elegibilidad y membresía

Enunciado

COMO

usuario con o sin membresía Círculo de Afiliados

QUIERO

ver recomendaciones que se adapten a mi nivel de membresía y elegibilidad real

PARA

acceder a los servicios que efectivamente puedo contratar según mi estado actual

Criterios de Aceptación
  • Motor consulta estado de membresía en tiempo real
  • Servicios exclusivos para Círculo se muestran solo a miembros
  • Para no miembros: muestra beneficios y ruta de afiliación
  • Elegibilidad validada contra criterios actualizados del catálogo
  • Indicador visual claro de servicios exclusivos vs. abiertos
Notas Técnicas
  • Integración con API de membresías
  • Cache de elegibilidad con TTL corto (≤ 15 min)
  • Degradación elegante si API de membresías no responde
