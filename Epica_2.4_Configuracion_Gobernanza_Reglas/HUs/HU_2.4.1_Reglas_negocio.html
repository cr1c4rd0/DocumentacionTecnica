Definir reglas de negocio

Enunciado

COMO

administrador de reglas del negocio

QUIERO

definir y gestionar reglas (listas blancas/negras, exclusiones, restricciones) que controlan las recomendaciones

PARA

asegurar que el motor siempre respete las políticas del negocio sin necesidad de redeploy

Criterios de Aceptación
  • Interfaz para crear/editar/activar/desactivar reglas
  • Tipos de regla: whitelist, blacklist, exclusión por segmento, restricción por fecha
  • Reglas tienen prioridad y orden de evaluación configurable
  • Cambios aplicados en la próxima inferencia sin redeploy
  • Historial de cambios con usuario y timestamp
  • Validación antes de activar para detectar conflictos
Notas Técnicas
  • Reglas almacenadas en YAML o base de datos
  • Motor de reglas desacoplado del modelo ML
  • Versionamiento de configuraciones
  • API para consultar reglas activas
