Historia de Usuario · HU 2.1.3

Aprende de mis clics y navegación

Enunciado

COMO

usuario que navega el portal

QUIERO

que el motor aprenda de mis clics, scroll y patrones de navegación en tiempo real

PARA

que mis recomendaciones mejoren progresivamente con cada interacción

Criterios de Aceptación
  • Frontend captura eventos (clic, scroll, tiempo en pantalla, navegación)
  • Eventos enviados al stream en ≤ 200ms
  • Pipeline procesa eventos y actualiza perfil del usuario
  • Embeddings del usuario se actualizan en ≤ 1 hora
  • Modelo ajusta pesos y preferencias dinámicamente
Notas Técnicas
  • Event stream con Kafka o Kinesis
  • Pipeline batch nocturno o streaming continuo
  • Feature store para embeddings
  • SDK de tracking con consentimiento GDPR
