HU 2.5.1 — Resolución de identidad omnicanal

El sistema unifica la identidad del usuario a través de canales (email, teléfono, IDs, cookies) generando un ID unificado con trazabilidad.

HU 2.5.2 — Identificación mínima y progresiva

Cuando no se puede identificar al usuario, el sistema activa perfilamiento mínimo (2 preguntas) para generar recomendaciones iniciales.

Ver detalle →

HU 2.5.3 — Respeta mis preferencias

El motor respeta las preferencias de comunicación y contenido del usuario, excluyendo servicios no deseados.

Ver detalle →

HU 2.5.4 — Preferencias y privacidad en UI

La UI permite al usuario gestionar sus preferencias de privacidad, consentimiento y canales de comunicación.
