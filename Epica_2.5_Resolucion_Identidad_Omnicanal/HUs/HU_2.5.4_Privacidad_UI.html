Historia de Usuario · HU 2.5.4

Preferencias y privacidad en UI

Enunciado

COMO

usuario del portal

QUIERO

gestionar mis preferencias de privacidad, consentimiento y canales desde la interfaz

PARA

tener control total sobre cómo se usan mis datos y cómo me contactan

Criterios de Aceptación
  • Panel de privacidad accesible desde perfil de usuario
  • Opciones de consentimiento granulares (personalización, comunicaciones, terceros)
  • Descarga de mis datos en ≤ 72 horas
  • Solicitud de borrado con confirmación y seguimiento
  • UI compatible con WCAG 2.1 AA
Notas Técnicas
  • Integración con Consent Management Platform (CMP)
  • API de derechos ARCO
  • Logs de consentimiento inmutables
  • Flujo de borrado coordinado entre sistemas
