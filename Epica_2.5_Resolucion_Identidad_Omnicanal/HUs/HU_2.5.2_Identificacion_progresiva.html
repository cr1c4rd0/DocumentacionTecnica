Historia de Usuario · HU 2.5.2

Identificación mínima y progresiva

Enunciado

COMO

usuario anónimo o parcialmente identificado

QUIERO

recibir recomendaciones relevantes con la mínima información que puedo compartir

PARA

obtener valor del sistema sin exponer más datos de los necesarios

Criterios de Aceptación
  • Máximo 2 preguntas de perfilamiento inicial
  • Recomendaciones generadas con datos mínimos disponibles
  • Mejora progresiva conforme el usuario comparte más información
  • Transición fluida de anónimo a identificado
  • Datos mínimos nunca se combinan sin consentimiento explícito
Notas Técnicas
  • Sesiones anónimas con ID temporal
  • Progressive profiling con consentimiento
  • Modelo degradado para usuarios poco conocidos
